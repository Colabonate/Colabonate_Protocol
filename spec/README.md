# spec/ — Ratified Specifications

This directory contains protocol specifications that have passed a milestone review and are considered stable for implementers to build against.

**Currently empty.** The first documents will move here at `v0.2.0`.

Documents are promoted from `docs/protocols/` to `spec/` when:
1. The document is complete (no `[PLANNED]` sections)
2. The relevant GitHub Milestone has been closed
3. No blocking open questions remain
4. At least one community review round has completed

See [SPECIFICATION_STATUS.md](../SPECIFICATION_STATUS.md) for the full status of all documents and promotion criteria.
