# Vision – Colabonate

**Normativity:** Descriptive

**Date:** 2026-03-21
**Status:** [IMPLEMENTED] Core marketplace | [ROADMAP] DAO, Governance

## Core Vision

Colabonate is a **decentralized marketplace and cooperation platform**
built on Bitcoin Lightning and Nostr.

The goal: enable people to collaborate economically — without a central authority,
without a bank account, without a trusted intermediary.

## Three Core Principles

**1. Sovereignty over control**
No central operator can ban accounts, block payments, or delete data.
Identity = your Lightning wallet (LNURL-Auth, pubkey-based).

**2. Bitcoin as the language**
Payments, escrow, reputation — all on Bitcoin Lightning. No altcoins,
no EVM, no bridging risks.

**3. Protocol over platform**
Instead of a monolithic platform: open protocols anyone can implement.
Like email, but for economic cooperation.

## What Colabonate Is (MVP — Phase 1)

A marketplace where:
- Offers are created and published via Nostr [IMPLEMENTED]
- Buyers and sellers interact via the ticket system [IMPLEMENTED]
- Payments are processed via Lightning [IMPLEMENTED] (mock-capable)
- Identity via LNURL-Auth — no password, no email [IMPLEMENTED]

## What Colabonate Will Become (Roadmap)

| Phase | Feature |
|-------|---------|
| Phase 2 | Lightning Escrow (multi-stage payments) |
| Phase 2 | Milestone-based cooperations |
| Phase 3 | Reputation system via Nostr Events (Soulbound) |
| Phase 4 | Dispute resolution (3-level) |
| Phase 4+ | DAO governance via Nostr votes |
| Phase 5 | Workflow editor + protocol library |

## What Colabonate Is Not

- Not an Ethereum / Solana / EVM project
- Not an NFT marketplace
- Not a custody service (keys stay with the user)
- Not KYC (a Lightning wallet is sufficient for identity)
- Not a centralized database as a single point of failure (goal: Nostr as P2P layer)

## Target Audiences

- Local merchants and service providers
- Freelancers and clients
- Cooperation partners and project teams
- Bitcoin / Lightning users who want to trade in a decentralized way
